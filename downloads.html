Muat Turun

Mula main

Adakah anda pengguna baharu? Silakan lihat halaman Mula Main, Soalan Lazim, dan Tutorial di wiki kami.
Kalau tidak kisah, juga lihat log perubahan.

Anda juga mungkin berminat untuk melihat beberapa permainan. Permainan menyediakan gaya main asas untuk enjin tersebut jalankan menggunakan skrip Lua. Permainan yang berbeza mempunyai matlamat yang berbeza, seperti kemandirian, pembinaan, atau Pemain lwn. Pemain. Biasanya Minetest didatangkan dengan permainan bernama Minetest Game, yang membekalkan item-item dan blok-blok lalai. Kemudiannya anda boleh menambah mods ke atas sesuatu permainan untuk menyesuaikan dengan pengalaman anda dengan lebih lagi.

Windows

Berfungsi di Windows 7, 8, 8.1 dan 10. Mungkin berfungsi di versi terdahulu, tetapi tidak disokong secara rasminya.

  • Minetest 5.2.0 - mudah-alih, 64-bit (disyorkan)
  • Minetest 5.2.0 - mudah-alih, 32-bit

Tersangkut? Sila lihat bantuan untuk mendapatkan Minetest di Windows.

Anda juga boleh dapatkan Minetest versi pembangunan terbaharu daripada binaan yang dicipta oleh ahli komuniti. Binaan ini lebih baharu daripada binaan yang diterbitkan secara rasmi dan mempunyai ciri-ciri baharu (dengan mengorbankan kestabilan).

Linux

Pakej

Ia mungkin ketinggalan zaman pada kebanyakan waktunya. Jika ia terlalu ketinggalan zaman, anda boleh bina daripada sumber.

  • Flatpak (semua edaran) - Stabil
  • Debian - Stabil
  • Ubuntu - Stabil
  • Ubuntu / Linux Mint (PPA) - Stabil, Tidak stabil
  • Fedora - Stabil
  • openSUSE - Stabil
  • Mageia - Stabil
  • Arch Linux - Stabil, Tidak stabil
  • Gentoo - Stabil
  • VoidLinux - Stabil

FreeBSD

Pengurus pakej

Di FreeBSD 9.1 dan terkemudian, anda boleh gunakan pakej pra-terbina rasmi: pkg install minetest minetest_game

Kompil menggunakan port

Anda boleh mengompil Minetest dan pilih pilihan binaan dengan mudah menggunakan dialog port FreeBSD.

cd /usr/ports/games/minetest
make install
cd /usr/ports/games/minetest_game
make install

macOS

macOS 10.10 atau terkemudian disyorkan.

  • Gunakan Homebrew untuk mendapatkan versi stabil:
    brew install minetest
  • Binaan harian tidak stabil

Kod sumber

Dapatkan kod sumber stabil terbaharu atau dalam pembangunan daripada GitHub.
Anda mungkin inginkan juga permainan Minetest Game. Letakkan permainan tersebut di direktori permainan anda.

Sila lihat README.md (bahasa Inggeris) untuk maklumat lanjut mengenai cara untuk mengompil Minetest daripada sumber.
