## Ciri-Ciri

- Menggunakan rangka kerja CSS [Bulma](https://bulma.io/).
- Menggunakan kaedah rekaan Web moden: unit `rem`, warna `hsl`.
- Laman dan favikon yang responsif.

## Aliran Kerja

Laman ini menggunakan penjana laman statik [Jekyll](https://jekyllrb.com).

- Pastikan anda memasangkan [Ruby](https://www.ruby-lang.org/) versi 2.4 atau lebih baharu.
- Pasangkan [Bundler](https://bundler.io/) dengan menjalankan `gem install bundler`.
  - Di Linux, gunakan pengurus versi Ruby seperti [rbenv](https://github.com/rbenv/rbenv)
    untuk menukar dengan mudah ke mana-mana versi dan elakkan isu-isu kebenaran.
- Pasangkan kebergantungan dengan menjalankan `bundle install`.
- Gunakan `bundle exec jekyll serve` untuk membina dan melayan laman secara tempatan dengan automatiknya.
